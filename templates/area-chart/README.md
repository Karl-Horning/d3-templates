# Area chart

An area chart showing Apple (AAPL) stock closing prices over time. Based on the [D3 area chart example](https://observablehq.com/@d3/area-chart/2) on Observable.

![Area chart showing Apple stock closing prices over time](../../imgs/area-chart-preview.png)

## Files

- `area-chart.html`
- `area-chart.js`—D3 implementation
- `data.js`—AAPL dataset
- `d3.v7.min.js`
